About Agora Blog

Project Overview

Agora Blog is a simple blog and social platform created as part of a software development portfolio. This project demonstrates modern web development practices, secure authentication, and social networking features.

The platform allows users to create accounts, write blog posts, follow other users, and interact within a secure, authenticated environment.

Key Features

User Authentication: Secure signup and signin with password validation

Two-Factor Authentication: Optional 2FA using TOTP (compatible with Google Authenticator, Authy, and Microsoft Authenticator)

Social Features: Follow/unfollow users, view profiles, and manage your network

Blog Posts: Create, view, and manage blog content

Security: CAPTCHA protection, secure sessions, HTTPS enforcement, and OWASP security headers

Technology Stack

Backend: Django 5.2.7 (Python)

Database: PostgreSQL (Production) / SQLite (Development)

Deployment: Heroku with Gunicorn

Security: django-otp, django-simple-captcha, whitenoise
